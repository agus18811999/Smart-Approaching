User Guideline & Best Practices

Version 1.0 • Smart Approaching System

1. Validasi Data Input

  • Format Nomor WhatsApp: Wajib menggunakan kode negara di depan tanpa simbol (+). Contoh: 62812345678. Sistem akan mencoba membersihkan karakter non-angka, namun input manual yang benar mempercepat proses.
  • Validasi Email: Pastikan alamat email memiliki format lengkap (user@domain.com). Kesalahan pengetikan akan menyebabkan tab Gmail terbuka dengan pesan error.

2. Langkah-Langkah Operasional

  1. Pilih mode approach (WhatsApp atau Email) melalui card di dashboard utama.
  2. Tentukan "Number of Leads". Masukkan angka sesuai jumlah target Anda.
  3. Klik "Generate" untuk memunculkan kolom input dinamis.
  4. Gunakan fitur Copy-Paste untuk mempercepat pengisian data dari spreadsheet Anda.
  5. Klik "Deploy Campaign". Browser akan mulai membuka tab satu per satu secara otomatis.

3. Larangan & Limitasi (PENTING)

Limitasi Broadcast: Sangat disarankan untuk tidak mengirim lebih dari 40 pesan dalam satu sesi deploy. Pengiriman masal yang terlalu cepat dapat dideteksi oleh WhatsApp sebagai aktivitas bot/spam.

Jeda Waktu: Sistem kami sudah menyertakan jeda (delay) otomatis. Jangan menutup tab utama sebelum seluruh tab target terbuka sempurna.

Konten Pesan: Hindari mengirim pesan yang sama persis secara berulang tanpa modifikasi (seperti nama partner). Personalisasi adalah kunci agar nomor Anda tetap aman (Anti-Ban).

Pop-up Blocker: Jika tab tidak terbuka, pastikan Anda sudah mengaktifkan izin "Always allow pop-ups" di pojok kanan atas address bar browser Anda.
